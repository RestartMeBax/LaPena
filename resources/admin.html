Image To Map

Simple version of your script. Upload an image, tune the three thresholds, then use the guided fields below to fill map entry data automatically.

This generates the terrain PNG input for map-generator/assets/maps/<map_name>/image.png. Then run Go map-generator and set Map Data URL to the output folder containing manifest.json and map.bin. Recommended source size is around 2-3 million pixels area (for example 2048x1024 or 2304x1296), with width and height divisible by 4. Preview uploads are automatically scaled/compressed for reliability.
